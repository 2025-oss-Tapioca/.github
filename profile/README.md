> **Tapioca는 MCP 기반의 통합 웹 개발 지원 도구입니다.** </br>
>API 명세 작성, erd 작성과 entity 생성, 코드 스캐폴딩, 트래픽 테스트, 성능 분석, 로그 모니터링까지 모든 기능을 제공합니다.</br>
>사용자가 서버를 연결하면 개발 초기부터 테스트, 운영에 이르기까지 전 과정을 하나의 환경에서 효율적으로 관리할 수 있도록 돕습니다.</br>

## 🔑 Key Features

<img width="1360" height="974" alt="Image" src="https://github.com/user-attachments/assets/b2cf768e-1204-4d16-a75f-c24a04ae685c" />

- **트래픽 테스트**  
  고부하 상황을 시뮬레이션하여 성능과 안정성을 검증할 수 있습니다.

<img width="1200" height="852" alt="Image" src="https://github.com/user-attachments/assets/f9bbe4a3-f4fc-4c8f-8aeb-3d64682cc899" />

- **성능 프로파일링**  
  실행 중 병목 지점을 파악하고 리소스 사용량을 분석할 수 있습니다.

<img width="1360" height="974" alt="Image" src="https://github.com/user-attachments/assets/b2cf768e-1204-4d16-a75f-c24a04ae685c" />

- **코드 스캐폴딩**  
  프로젝트의 기본 코드 구조를 자동으로 생성해줍니다.

<img width="1608" height="1148" alt="Image" src="https://github.com/user-attachments/assets/49fb5b37-090f-40c8-ac5e-463d3311578e" />
  
- **API 명세 관리**  
   API 명세를 작성해주며 작성 후에는 명세를 기록합니다.

<img width="1360" height="974" alt="Image" src="https://github.com/user-attachments/assets/b2cf768e-1204-4d16-a75f-c24a04ae685c" />

- **API 목업**  
  백엔드 없이도 프론트엔드 개발 및 테스트가 가능합니다.

<img width="1210" height="858" alt="Image" src="https://github.com/user-attachments/assets/5341f514-ad43-4fca-a350-860d4776f55a" />

- **로그 모니터링**  
  MCP를 통해 서버 로그를 실시간으로 수집하고 시각화할 수 있습니다.

<img width="1590" height="1128" alt="Image" src="https://github.com/user-attachments/assets/79a139f9-837b-45a0-b1d6-a006b74c6bad" />

- **erd 작성과 entitiy 생성**  
  ERD를 입력하면, 백엔드에서 Entity 클래스를 자동 생성합니다.
